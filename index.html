📱 Cài đặt như app

Trên iPhone (Safari):

1. Bấm nút Share (hình mũi tên lên)
2. Cuộn xuống chọn "Add to Home Screen"
3. Đặt tên (VD: "XS 6/55") → bấm Add
4. Icon xuất hiện trên màn hình chính như app thật

Trên Android (Chrome):

1. Bấm menu 3 chấm góc phải trên
2. Chọn "Add to Home screen"
3. Đặt tên → Add

✅ Sau khi cài, app chạy offline 100%, dữ liệu lưu vĩnh viễn.

🔄 Đồng bộ giữa thiết bị

Vì app chạy local, mỗi thiết bị có dữ liệu riêng. Để đồng bộ:

Cách 1 — Export/Import file:

Trên thiết bị A: bấm Export → lưu file JSON.
Gửi file qua email/messenger sang thiết bị B.
Trên thiết bị B: bấm Import → chọn file đó.

Cách 2 — Copy code đồng bộ:

Dữ liệu được nén thành text base64 ngắn → copy text này → paste vào thiết bị khác.
